SimNIBS installer
=================

This program installs and updates SimNBIS (www.simnibs.org)

To start the GUI:

```
./install_simnibs
```

To get help:
```
./install_simnibs -h
```

Install in silent mode (no GUI, will accept licenses and do all install steps)
```
./install_simnibs -s
```

More information
-----------------
Please visit www.simnibs.org
contact: support@simnibs.org
